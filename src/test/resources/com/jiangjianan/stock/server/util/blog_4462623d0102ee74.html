坐怀不乱

(2013-01-12 09:51:46)
转载▼
标签：

杂谈

 分类： 投资札记

　　美色当前，最大的挑战是：坐怀不乱。

　　男人的体力是有限的，听说过累死的耕牛，没听说过犁坏的地。

　　真男人，必须要学会在一块好地上用心耕耘。

 

　　投资，很大程度上，是一个长期说“不”的过程。

　　肯定自己，是非常容易的；否定自己，相对较难。

　　容易做到的事，注定不会是非常事。

 

　　持币观望、重仓押注，是两个极端。

　　坐怀不乱的本性，在投资赌局上，是绝对的极端个性。

 

　　对美色的猎艳，前戏一定要做的很充分：按年调戏。

　　当高潮即将到来时，才可以全力压上，这样会降低体能不足的先天缺陷。

　　牛，就是这样了，力量有余，耐力不足。

 

　　整天进进出出，肯定是消耗体力的。

　　偶尔，来那么几下，效果一定会不一样。

 

　　坐怀不乱，实际上是假正经。

　　内心深处，大多都是急切的一塌糊涂。

 

　　作为老色鬼，从急色鬼进化成功之后，就深刻的明白一个道理：美女遍地，绝色唯一。

　　绝色，靠等，靠捂，不靠弄。

 

　　该装的时候，还是要装的。

　　可以调戏，但不可乱来。
